Instalar como aplicación de escritorio
Opción 1 — Chrome / Edge (recomendado):
1. Abre este archivo con un servidor local:
python -m http.server 8080 en la carpeta del archivo
2. Ve a http://localhost:8080/SIGAC.html
3. En Chrome: menú ⋮ → "Instalar SIGAC..."
4. ¡Aparece en tu escritorio como app nativa!
Opción 2 — Acceso directo rápido:
1. Abre el archivo en Chrome
2. En la barra de direcciones → ícono ☆ → "Crear acceso directo"
3. Marca "Abrir como ventana" → Crear
Opción 3 — Hosting gratuito (acceso desde cualquier lugar):
Sube el archivo a netlify.com/drop (arrastra el archivo) → URL pública HTTPS → instala como PWA en cualquier dispositivo.
